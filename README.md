## 🔍 What leetha does

leetha is a passive network fingerprinting and analysis tool for Windows. It watches network traffic and helps you see what devices, services, and patterns appear on your network.

Use it when you want a clearer view of network activity without setting up a heavy tool chain. It gives you a simple way to inspect traffic and collect fingerprints from live network data.

## 💻 What you need

Before you install leetha, make sure your PC meets these basic needs:

- Windows 10 or Windows 11
- An internet connection for the first download
- Admin access on your PC
- At least 200 MB of free disk space
- A modern 64-bit Intel or AMD processor
- A network adapter that can capture local traffic

For best results, close other network tools while you run leetha.

## 🧰 Troubleshooting

### The app does not open

- Check that you downloaded the full file
- Extract the ZIP folder if the download came compressed
- Run the app as administrator
- Make sure Windows did not block the file

### No devices appear

- Pick the correct network adapter
- Check that your PC is connected to a network
- Wait a few minutes for traffic to build
- Restart the capture session

### Results look empty

- Increase scan time
- Use a different adapter
- Make sure other security tools are not filtering traffic
- Try again after normal network use begins

### Windows asks for extra permission

- Choose Yes
- Open the app again with admin rights
- Check your Windows security settings if the prompt returns
